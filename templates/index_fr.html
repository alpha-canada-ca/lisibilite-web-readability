Comment utiliser cet outil

Entrez une URL de Canada.ca pour obtenir un score de lisibilité Web.

Il s'agit d'un outil expérimental.. Ne l'utilisez pas comme une mesure précise de la lisibilité ou pour évaluer la conformité à un indicateur de performance.

Il est conçu pour donner une idée générale de la lisibilité d'une page sur le web. Un bon score ne garantit pas que la page soit conviviale ou utile.

Cet outil utilise le texte qui figure dans la balise main de la page HTML. Ceci peut inclure du texte qui n'est pas visible pour les gens. L'outil exclut le texte qui se trouve dans un tableau pour les besoins du calcul.

Voir le code et la méthodologie sur GitHub:

https://github.com/alpha-canada-ca/lisibilite-web-readability
